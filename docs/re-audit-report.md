## 六、问题分级汇总

### P0（阻塞性问题）
1. **订单接口认证错误**：GET /api/v1/orders 返回 401，导致订单流程无法完成
2. **数据库依赖**：后端服务依赖 MySQL 数据库，本地环境可能无法连接

### P1（重要问题）
1. **管理后台 Dashboard**：使用 mock 数据，未与真实 API 对接
2. **前端购物车**：使用本地 store，未调用后端 API
3. **后端 deps.py 语法错误**：第 74 行使用了错误的语法

### P2（一般问题）
1. **商品详情页加入购物车**：使用本地 store，未调用后端 API
2. **管理后台 token 字段不一致**：部分使用 `token`，部分使用 `access_token`
3. **前端商城商品搜索**：搜索功能未实现

### P3（增强问题）
1. **前端错误处理**：缺少统一的错误处理机制
2. **后端 API 文档**：需要完善 API 文档
3. **测试覆盖**：缺少单元测试和集成测试
