# 开发缺口清单

本文档列出了项目当前存在的所有缺口和需要完成的工作。

---

## P0 - 阻塞性问题（必须优先解决）

### 1. 前端商城路由缺失
- **问题**: 前端商城缺少多个关键路由
- **影响**: 用户无法访问商品列表、商品详情、登录、注册等页面
- **缺口**:
  - 商品列表页路由 (`/products`)
  - 商品详情页路由 (`/product/:id`)
  - 登录页路由 (`/login`)
  - 注册页路由 (`/register`)
  - 个人中心页路由 (`/profile`)
- **代码位置**: `frontend-shop/src/router/index.ts`
- **优先级**: P0

### 2. 前后端 API 路径不匹配
- **问题**: 前端商城调用的 API 路径与后端不匹配
- **影响**: 首页、商品列表等无法正常加载数据
- **缺口**:
  - 前端调用 `/api/v1/shop-products/shop/` 但后端没有该路由
  - 需要统一 API 路径规范
- **代码位置**: `frontend-shop/src/api/product.ts`, `backend/app/api/v1/api.py`
- **优先级**: P0

### 3. 用户注册功能缺失
- **问题**: 没有用户注册 API 和页面
- **影响**: 新用户无法注册
- **缺口**:
  - 后端注册 API (`POST /api/v1/auth/register`)
  - 前端商城注册页
  - 管理后台用户创建功能
- **代码位置**: `backend/app/api/v1/auth.py`, `frontend-shop/src/views/`
- **优先级**: P0

### 4. 数据库初始化和数据填充
- **问题**: 需要确保数据库可以正常初始化和填充测试数据
- **影响**: 项目无法正常运行
- **缺口**:
  - 测试并完善 `init_db.py`
  - 测试并完善 `seed_data.py`
  - 确保 Alembic 迁移可用
- **代码位置**: `backend/scripts/`
- **优先级**: P0

### 5. 后端 API 路由未全部注册
- **问题**: 大量 API 路由文件存在但未在 api.py 中注册
- **影响**: 分类、用户、仪表盘等 API 无法访问
- **缺口**:
  - 注册 categories 路由
  - 注册 users 路由
  - 注册 dashboard 路由
  - 注册其他未注册的路由
- **代码位置**: `backend/app/api/v1/api.py`
- **优先级**: P0
