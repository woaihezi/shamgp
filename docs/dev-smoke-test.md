## 7. 常见问题排查

### 问题：后端启动失败
**排查步骤：**
1. 检查端口 8000 是否被占用
2. 检查虚拟环境是否激活
3. 检查依赖是否安装完整：`pip list`
4. 查看启动日志中的错误信息

### 问题：登录失败
**排查步骤：**
1. 确认后端服务正常运行
2. 确认数据库已初始化并包含种子数据
3. 检查用户名密码是否正确
4. 查看浏览器 Network 标签中的请求详情

### 问题：前端页面空白
**排查步骤：**
1. 查看浏览器控制台是否有错误
2. 检查 npm install 是否成功
3. 检查 npm run dev 的输出日志
4. 确认 vite.config.ts 中的代理配置正确

### 问题：API 请求失败
**排查步骤：**
1. 确认后端服务运行在 8000 端口
2. 检查前端请求是否通过 proxy 发送
3. 查看 Network 标签中的请求和响应
4. 确认 CORS 配置正确（后端已允许所有来源）
